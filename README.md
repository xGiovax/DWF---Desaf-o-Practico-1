# Sistema Escolar -- Desafío Spring Boot

Proyecto desarrollado con Spring Boot, JPA y H2 que gestiona alumnos y
materias, incluyendo su relación muchos-a-muchos.

------------------------------------------------------------------------

## Tecnologías utilizadas

-   Java 17+
-   Spring Boot
-   Spring Data JPA
-   H2 Database (en memoria)
-   JUnit 5
-   Maven
